# 📂 Projects

This repository contains the following projects completed during the internship:

### 1️⃣ Create EBS Snapshot
Learned how to create snapshots of EBS volumes to enable backup, restore, and disaster recovery capabilities.

### 2️⃣ Create EC2 Instance
Provisioned and configured a virtual machine instance on AWS EC2 for application hosting.

### 3️⃣ Create IAM Users
Configured IAM users, groups, and policies to implement secure access control using the **least privilege principle**.

### 4️⃣ Create S3 Bucket
Created and configured an Amazon S3 bucket for object storage and data management.

### 5️⃣ Create VPC
Designed a Virtual Private Cloud with custom networking components including subnets, routing, and internet connectivity.

### 6️⃣ Deploy Web Server on EC2
Deployed and configured a web server on an EC2 instance to host a simple application.

### 7️⃣ Host Static Website
Hosted a static website using **Amazon S3 static website hosting**.

### 8️⃣ Monitoring and Logs using CloudWatch
Configured monitoring and logging for cloud resources using **Amazon CloudWatch**.
